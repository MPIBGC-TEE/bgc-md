About the model

The model depicted in this document considers carbon allocation with a process based approach. It was originally described by Running & Coughlan (1988).

Input fluxes

\(C_{f}: \eta_{f}\cdot u\)
\(C_{r}: \eta_{r}\cdot u\)
\(C_{w}: \eta_{w}\cdot u\)

Output fluxes

\(C_{f}: C_{f}\cdot\gamma_{f}\)
\(C_{r}: C_{r}\cdot\gamma_{r}\)
\(C_{w}: C_{w}\cdot\gamma_{w}\)

Steady state formulas

\(C_f = \frac{\eta_{f}\cdot u}{\gamma_{f}}\)

\(C_r = \frac{\eta_{r}\cdot u}{\gamma_{r}}\)

\(C_w = \frac{\eta_{w}\cdot u}{\gamma_{w}}\)
